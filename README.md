# Solar Project

A travel booking site that allows users to browse and book trips to space.

# Link

https://solar-project-2022.herokuapp.com/

## Features

1. Registration (with welcome email)
2. Log in and log out
3. Browse available destinations
4. Add to cart
5. Delete from cart
6. Edit profile
